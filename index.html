WebGL RPG / Hack & Slash

This demo has only been tested with chrome 10. It may work with older versions of chrome, and some features do not work with firefox 4.0 (skill drag & drop). You will probably need to download a beta of a recent browser to see this demo: getting a WebGL Implementation. If you need information about this demo, you can contact me : nicolas dot bonnel at gmail dot com. The code is licenced under GNU GPL 3. Feel free to reuse the code, credits would be appreciated.

Controls

For the moment, controls are hardcoded. No WASD because I use a french keyboard layout :).
  • Forward : E
  • Backward : D
  • Turn left : S
  • Turn right : F
  • Strafe left : Z
  • Strafe right : R
  • Attack : T
  • Grab loot : Q
  • Interact (open, talk): A
  • Pause : P
  • Show/hide aptitude panel : L
  • Show/hide inventory panel : I
  • Show/hide settings panel : O
